# Sleep Tracker Instructions

## How to Use This Tracker

1. **Fill in daily** - Every morning, log yesterday night's sleep
2. **Be consistent** - Track for at least 2 weeks to see patterns
3. **Use the Notes column** - Track factors like caffeine, stress, exercise, etc.
4. **Review weekly** - Look for trends and what affects your sleep

## Column Definitions

| Column | Description |
|--------|-------------|
| **Date** | The day you woke up |
| **Bedtime** | When you got in bed to sleep (e.g., 10:30 PM) |
| **Wake Time** | When you got out of bed (e.g., 6:30 AM) |
| **Sleep Duration** | Total hours asleep (e.g., 7.5) |
| **Time to Fall Asleep** | Minutes it took to drift off |
| **Wake-ups** | Number of times you woke during the night |
| **Sleep Quality** | Rate 1-10 (10 = best sleep ever) |
| **Energy Level** | Rate 1-10 (10 = fully energized all day) |
| **Notes** | Track anything affecting sleep |

## Tips for Better Tracking

- Keep this file on your phone or computer
- Set a morning reminder to fill in data
- Be honest with ratings - this is for YOU
- Look for patterns after 2+ weeks

## Goals to Aim For

- **Sleep Duration:** 7-9 hours
- **Sleep Quality:** 7+
- **Energy Level:** 7+
- **Time to Fall Asleep:** < 20 minutes
- **Wake-ups:** 0-1 per night
- **Bedtime:** 10:00-11:00 PM
- **Wake Time:** 6:00-7:00 AM
